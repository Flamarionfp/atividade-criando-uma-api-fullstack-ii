### Como executar o projeto?

Instalar as dependências

```bash
npm install
```

Criar as tabelas necessárias

```bash
npm run migrate
```

Executar o servidor de desenvolvimento

```bash
npm run dev
```

Subir a carga de dados iniciais

```bash
npm run seed
```

Executar o código de produção

```bash
npm run build

&&

npm start
```
